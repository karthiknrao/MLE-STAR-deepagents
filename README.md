# MLE-STAR Deep Agents

## Quick Start

```bash
python -m mlestar.run --task housing-price --data_dir mlestar/task
```

## Usage

```bash
# Run with a specific task and data directory
python -m mlestar.run --task <task_name> --data_dir <path>

# Run with a custom config
python -m mlestar.run --config config.yaml --task spaceship-titanic
```
